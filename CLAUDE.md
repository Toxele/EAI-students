# Описание проекта:
Проект: AI-анализ микрофотографий полированных шлифов руды

Цель

Разработать end-to-end систему, которая автоматически анализирует панорамные микрофотографии полированных шлифов руды и классифицирует образец.

Основная задача — заменить ручную работу геолога автоматическим CV/ML пайплайном.

Входные данные

Поддержать:

* TIFF
* PNG
* JPEG

Изображения могут быть очень большими (до **10000×10000 px**).

Pipeline

1. Предобработка

Автоматически выполнить:

* нормализацию освещения;
* коррекцию контраста;
* шумоподавление;
* работу с большими изображениями (тайлы/patches);
* сохранение геометрии объектов.

2. Сегментация

На уровне пикселей необходимо выделить:

Sulfide inclusions

Разделить на два класса:

* **Ordinary intergrowths**

  * крупные
  * почти не замещены нерудной фазой

* **Fine intergrowths**

  * значительно замещены нерудной фазой
  * труднообогатимая руда

Talc

Отдельный класс.

Тёмная рассеянная фаза внутри нерудной матрицы.

3. Количественный анализ

После сегментации вычислить:

* площадь сульфидов;
* площадь обычных срастаний;
* площадь тонких срастаний;
* площадь талька;
* процент каждого класса относительно изображения.

4. Правила классификации

Использовать экспертную логику.

```
if talc_percent > 10%:
    ore = "Talc ore"

else:
    if ordinary_percent > fine_percent:
        ore = "Ordinary ore"
    else:
        ore = "Hard-to-beneficiate ore"
```

Не использовать black-box классификатор без объяснимой логики.

Выходные данные

Цветная маска

Overlay поверх исходного изображения:

* green — ordinary intergrowths
* red — fine intergrowths
* blue — talc

Метрики

Отобразить:

* total sulfides %
* ordinary %
* fine %
* talc %

Итоговое заключение

Пример:

```
Ore type: Talc ore

Talc: 14.2%

Fine intergrowths: 62%

Ordinary intergrowths: 38%
```
